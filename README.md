# About
this is port of gruber-darker colorscheme to kitty terminal emulator

# Screenshot

# Installation
copy `gruber.conf` to your `~/.config/kitty` and put that line in your kitty.conf
```
include ./gruber.conf
```

Note: `kitty.conf` should be located at `~/.config/kitty`
